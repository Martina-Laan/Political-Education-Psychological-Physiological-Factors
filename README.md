# Political-Education-Influence-on-Students-Psychological-and-Physiological-State

Topic: Political Education Influence on Students' Psychological and Physiological State

Question 1: How do the psychological and physiological parameters change depending on stimuli?

Question 2: Are psychological and physiological reactions coherent with one another?

Question 3: How do independent variables (such as age, gender, education level) affect the reactions?

Methods: Data visualization, correlation, clustering, linear regression models, creating a random forest model, hyperparameter tuning etc.

Dataset URL: https://www.kaggle.com/datasets/ziya07/psychological-state-identification-dataset/data

Dataset (256.74 kB): Dataset that consists of biosensor-collected physiological, behavioral, environmental data and psychological states of individuals, particularly students, during ideological and political education (IPE) activities.
